# SkillPulse v1.0.0 — Public Launch

**Released:** 2026-04-24

## What's included

### Data Pipeline
- **7 ingestors**: GitHub Search API, npm registry, PyPI, Anthropic MCP Registry, Tokrepo, Hacker News, Reddit
- **AI classifier**: Claude Haiku (haiku-4-5) with Sonnet (sonnet-4-6) escalation for low-confidence entries
- **22 canonical categories** — no AI-invented taxonomy drift
- **Deduplication**: URL normalization + fuzzy author+name matching
- **Pulse Score** (0–100): stars, growth rate, recency, cross-source, confidence

### Site
- Astro static site deployed to GitHub Pages
- Dark theme with MC. brand palette (#0F0F0F / #D4882A / #C9A84C)
- Fuzzy search (Fuse.js) + category filter
- Keyboard shortcut `/` to focus search
- Pages: index, /all, /about, /stats
- SEO: Open Graph meta, sitemap.xml, robots.txt

### Automation
- GitHub Actions refresh every 6 hours (cron)
- Automatic README rebuild on each refresh
- Issue bot: submission queuing, duplicate detection, stale URL verification
- Weekly digest (Mondays 12 UTC)
- Budget guard: aborts classifier if monthly Anthropic spend exceeds $50

### Developer Experience
- TypeScript strict mode throughout
- pnpm monorepo with 5 packages
- Biome for linting and formatting (single config)
- Vitest with 24 tests
- Dependabot for automated dependency updates
