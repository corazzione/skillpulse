## How It Works

1. **Ingest** — Scrapers pull data from GitHub, npm, PyPI, HN, Reddit, and Anthropic Registry every 6 hours
2. **Classify** — Claude AI classifies each entry, assigns category, compatibility, and pulse score
3. **Publish** — This README and the site are automatically rebuilt and pushed
