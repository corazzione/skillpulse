# Product Hunt Launch

**Tagline:** Auto-updated registry of AI agent skills, MCPs & prompts — refreshed every 6h

**Description:**
SkillPulse is a living registry of Claude Code skills, MCP servers, Cursor plugins, and AI agent tools — updated every 6 hours automatically.

No more stale awesome lists. A bot scrapes 7 sources (GitHub, npm, PyPI, HN, Reddit, Anthropic Registry), Claude AI classifies each entry, and a Pulse Score ranks everything by stars, growth rate, and recency.

Submit a skill via GitHub issue → bot validates it → it appears in the registry within 6 hours.
